# agropasz-website

Strona statyczna AGROPASZ GROUP (wersja początkowa).

Pliki w repo:
- index.html
- styles.css
- logo.svg

Instrukcje:
1. Wgrać pliki do branch `main`.
2. Aby opublikować jako GitHub Pages, w ustawieniach repo ustaw Pages -> branch `main` / folder `/ (root)`.
3. Podmienić action formularza w sekcji kontaktu na własny endpoint (np. Formspree) oraz zaktualizować dane kontaktowe.
